Написать программу, помогающую студенту выбрать обед в столовой по следующему алгоритму:
- программа выводит на экран меню в формате:
Первые блюда:
1. наименование - цена
2. наименование - цена
3. наименование - цена
4. наименование - цена
Вторые блюда:
1. наименование - цена
2. наименование - цена
3. наименование - цена
4. наименование - цена
Напитки:
1. наименование - цена
2. наименование - цена
3. наименование - цена
4. наименование - цена
- пользователь вводит через запятую набор типов блюд, которые он хочет съесть: например, "первое, напиток" или "первое, второе, напиток";
- пользователь вводит сумму, которую он готов потратить на обед;
- программа выдает пользователю максимально дорогой набор блюд, укладывающийся в указанную сумму и соответствующий желаемому набору, и его стоимость;
- программа предлагает пользователю блюда, которые можно купить на оставшуюся сумму.
Типы блюд, их наименования и стоимость хранятся в тексте программы. Каждый запрос и выводимый параметр должен сопровождаться текстовым описанием. Программа должна обрабатывать возможные ошибки вводимых пользователем значений.

Решение необходимо оформить в виде файла .py, его нужно прикрепить здесь в Teams.
Приложенный файл должен выполняться интерпретатором Python без дополнительных манипуляций со стороны преподавателя.
